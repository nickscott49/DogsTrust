1Opening the app

Open the HTML file in any browser — Chrome, Edge, or Safari all work. You'll land on the Setup screen first.

  • Organisation name and Website URL are already filled in as "Dogs Trust" and "dogstrust.org.uk" — only change these if you need to.
  • Primary colour and Accent colour are already set to Dogs Trust's black and Sunshine Yellow. Most people can leave these alone.
  • Logo — the Dogs Trust logo is already loaded and built into the app. If you ever need a different one, click Remove and upload a replacement PNG or JPEG.

Click Launch Image Maker → to continue.

Already set up before? If you've saved a settings file previously, use Import saved settings instead — it restores your stories and style choices in one go. See Section 7.

2Adding a rehoming story

The Rehoming Stories panel (top-left of the app) is where you add each dog's story. There are three ways in.

Single

  1. Type or paste the dog's story into the text box.
  2. The dog's name is often detected automatically into the Dog's name field — check it and correct it if needed.
  3. Optional: click ♥ Add heart to end to add a heart to the end of the story (it's coloured to match your accent colour on the finished image).
  4. Click Add to list.

Bulk paste

Use this when you have several stories to add at once.

  1. Switch to the Bulk paste tab.
  2. Paste all your stories in, with a blank line between each one.
  3. Click Parse stories — each one gets its own preview card where you can fix the detected name.
  4. Click Add all … stories to list.

CSV file

Use this if your stories are already in a spreadsheet.

  1. Switch to the CSV file tab.
  2. Your file needs two columns: story text, dog's name. Click Download CSV template for a starting point.
  3. Upload your CSV, review the preview list (remove any row with ×), then click Add … stories to list.

Managing stories you've already added

Every story appears as a card in the list:

  • Click the story text to select it for the preview.
  • Edit the dog's name directly in its small field.
  • ✎ edits the story text. ♥ adds a heart to the end. × deletes it.

3Adding photos

In the Photos panel, click the upload area and choose one or more JPG or PNG photos. Thumbnails appear below — click one to make it active in the preview, or hover and click × to remove it.

4Styling the image

The Style panel controls how the finished image looks. Everything updates the preview live.

5Positioning the photo

In the Preview panel:

  • Drag the image to reposition it within the frame.
  • Scroll, or use the Zoom slider, to zoom in.
  • Use the arrow buttons for small nudges, or Reset to start over.

6Generating the image

Once you're happy with the preview:

  1. Click Generate save image.
  2. A full-resolution (1080×1920px) version appears below.
  3. Right-click it and choose "Save image as…", or drag it straight to your desktop.

Generating many images at once — Bulk generate

If you have multiple photos and multiple stories, use Bulk generate (top-right of the app).

Cycle

🔄 In order

Each photo gets a story in turn, looping back round if there are more photos than stories.

All combinations

⊞ Every pairing

Every photo paired with every story. Can produce a lot of images.

Random

🎲 Random pairing

Each photo is paired with a randomly chosen story.

Click Generate & preview all — each image gets a thumbnail with its own download arrow. Click Download all (…) to save every image at once.

7Saving your settings for next time

If you've customised colours or style settings, or added a batch of stories you'll want again, save all of it:

  • Click Export (top-right) to download a settings file (.json).
  • Next time, click Import (or Import saved settings on the Setup screen) and choose that file to restore everything instantly — including your stories and style choices.
Note The logo doesn't need saving or restoring — it's already built into the app file itself.

Tips

  • The dog's name field is mainly for organising your list — only the story text appears on the finished image, so it reads best when the name is written naturally into the sentence, e.g. "After eight months waiting, Biscuit has finally found his forever home."
  • If the story wraps onto several lines and looks cramped, try reducing Text size or switching Story position to Top or Centre.
  • Use Logo opacity together with Overlay darkness to keep the logo and text legible against busy or bright photos.
